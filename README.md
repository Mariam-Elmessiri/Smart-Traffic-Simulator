# Smart-Traffic-Simulator
A simple simulation system to manage traffic flow at intersections using Python and the `numpy` library

## Overview
This project simulate vehicle density in 4 different streets. The system analyzes traffic in real-time, identifies the most congested streets that need a "Green Light" and alerts the user if traffic leveks reach a critical point

### what this project does
* Tracks traffic in 4 streets.
* Identifies busy streets automatically.
* Shows alerts when treffic is very busy.
* Uses color-coded alert for better readability.

## How to Run
1. Install NumPy:
   ```bash
   pip install numpy
   ```
2. Run the script:
   ```bash
   python SmartTraffic.py
   ```
